# 环境准备
- Github账号，新库名:用户名.github.io
- node.js、npm、JS、Git 安装与基础学习
- 图床 Gitee+PicGo  

# 初步搭建
- 创建博客文件夹，shift+右键进入目录cmd
- 安装hexo
- npm install -g hexo-cli
- 初始化文件夹
- hexo init hexo_blog 
- 进入博客文件夹
- cd E:\my\hexo_blog
- 安装博客需要的依赖文件
- npm install  
- Hexo命令
- hexo cl	#清理
- hexo g	#生成
- hexo s	#本地服务 
- hexo d	#发布
- 测试http://locakhost:4000  或127.0.0.1:4000 
- 初步搭建完成
